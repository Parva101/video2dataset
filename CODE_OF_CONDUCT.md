# Code of Conduct

This project follows a Contributor Covenant style code of conduct.

## Our standards

Examples of behavior that contribute to a positive environment:

- Being respectful and constructive
- Giving and accepting feedback professionally
- Assuming good intent and focusing on technical outcomes

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or derogatory comments
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code,
issues, and other contributions that violate this Code of Conduct.

## Reporting

Report violations by opening a private security report or contacting the
maintainer through GitHub.

Maintainers will review and respond as promptly as possible.
